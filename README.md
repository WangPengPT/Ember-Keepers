# 余烬守卫 (Ember Keepers)

一款基于Unity 6.2开发的英雄防御Roguelite游戏

## 🎮 游戏简介

《余烬守卫》是一款波次防御Roguelite游戏，玩家在波次间隙管理和培养英雄，英雄在战斗中全自动作战。游戏结合了：
- 🔥 4种元素（火、冰、雷、土）的克制机制
- ⚔️ 12个可玩英雄（4元素×3职业）
- 🎯 全自动ARPG战斗
- 📊 深度的英雄培养系统
- 🔄 Roguelite元进度系统

## ✨ 核心特性

### 波次间隙管理
- 每波次结束后自动暂停
- 在策略阶段购买英雄、属性书、装备
- 部署和调整英雄位置
- 升级基地核心

### 全自动战斗
- 英雄自动移动、攻击、释放技能
- 玩家专注于策略规划
- 观察战局，使用基地主动技能

### 元素克制系统
- 火克冰，雷克土
- 不同元素怪物需要不同英雄对抗
- 元素专精影响伤害和抗性

### 元进度系统
- 使用星火精粹进行局外永久升级
- 英雄血脉强化
- 初始金币储备
- 商店优化
- 精粹获取效率

## 📁 项目结构

```
Game/Assets/Scripts/
├── Core/              # 核心管理器（GameManager, WaveManager等）
├── Data/              # 数据表和枚举定义
├── Heroes/            # 英雄系统（12个英雄+技能）
├── Equipment/         # 装备系统
├── Monsters/          # 怪物系统（普通怪+精英+Boss）
├── Wave/              # 波次生成系统
├── Combat/            # 战斗系统
├── Base/              # 基地核心系统
├── Shop/              # 商店系统
├── Deployment/        # 部署系统
├── MetaProgression/   # 元进度系统
└── UI/                # UI系统（10个UI脚本）
```

## 🎯 游戏内容

### 英雄（12个）
- **火系**：烬火射手、熔岩壁垒、炽热咏者
- **冰系**：冰晶狙击、极地卫士、寒霜祭司
- **雷系**：电弧法师、雷霆战甲、脉冲技师
- **土系**：碎岩者、大地之盾、泥沼巫师

### 波次
- **主线关卡**：20波（包含3个Boss）
- **无尽模式**：10波循环，难度持续递增

### Boss
- **第10波**：荒芜之主（多线召唤）
- **第15波**：永冻之灾（元素锁定）
- **第20波**：裂隙之心（技能压制）
