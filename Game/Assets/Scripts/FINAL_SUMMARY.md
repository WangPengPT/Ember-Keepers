## 📖 使用指南

### 游戏流程
1. **主菜单** → 开始新游戏
2. **策略阶段**（波次间隙）
   - 购买英雄、属性书、装备
   - 部署英雄到防御区域
   - 升级基地
3. **战斗阶段**
   - 英雄自动战斗
   - 观察战况
   - 使用基地主动技能（净化冲击）
4. **波次结束** → 返回策略阶段
5. **游戏结束** → 结算星火精粹 → 元进度升级

### 开发指南
- 数据配置：在DataManager中加载ScriptableObject数据
- 英雄创建：继承HeroBase，实现InitializeSkill方法
- 技能创建：继承HeroSkillBase，实现ExecuteSkill方法
- 怪物创建：继承MonsterBase，实现Initialize方法
- 装备创建：继承EquipmentBase，配置属性和词条
