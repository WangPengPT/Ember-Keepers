# ⚔️ 游戏设计文档：余烬守卫 (Ember Keepers) - **最终版**

## 🎯 核心玩法：波次间隙管理与全自动 ARPG 战斗

**《余烬守卫》** 是一款英雄防御 Roguelite 游戏。玩家在波次间隙管理和培养英雄，英雄在战斗中全自动作战。

| 元素 | 机制说明 | 策略深度 |
| :--- | :--- | :--- |
| **回合结构** | **怪物分波次涌入**。每波次结束后，游戏**自动暂停**，进入策略阶段。 | 玩家的决策（购买/升级/部署）时间是有限且宝贵的。 |
| **战斗核心** | 战斗一旦开始，全程**实时进行 (ARPG 模式)**。英雄**自动攻击**，并且**自动施放技能**。 | 玩家专注于英雄培养、装备搭配和位置调动，不进行战斗微操。 |
| **商店/暂停** | **只有在波次间隙暂停时，才可访问商店**、英雄面板、进行升级和防线调动。 | 所有策略必须在准备阶段完成。 |
| **失败条件** | 基地核心的“星火能量”耗尽（生命值归零）。 | 需时刻平衡输出和对核心的保护。 |

## 🌎 核心设定与世界观

### 游戏名称：**余烬守卫 (Ember Keepers)**

* **世界观：** 玩家的基地是世界上最后一座拥有古老**“星火”**能量的要塞。这股能量是阻挡混沌、维持世界理智的最后屏障。
* **敌人来源：** **“影界裂隙”**不断扩大，从中涌出以太生物、元素畸变体和混沌巨兽。
* **英雄设定：** 英雄是来自不同王国和种族的精英，他们拥有各自的**元素专精（火、冰、雷、土）**，才能有效对抗影界生物。

## 🌌 局外永久升级系统：星火元进度 (Meta-Progression)

该系统允许玩家使用局外资源——**“星火精粹”**（Starfire Essence，来自每局游戏的结算奖励）来永久性强化游戏的基础参数，让后续的游戏局更容易进行，或解锁新的内容。

| 升级项目 | 效果描述 | 资源消耗 | 策略价值 |
| :--- | :--- | :--- | :--- |
| **英雄血脉强化** | 永久提高所有英雄的**初始生命值**。 | 星火精粹 | 增强所有英雄的基础生存能力。 |
| **初始金币储备** | 永久提高玩家**每局游戏开始时获得的初始金币**数量。 | 星火精粹 | 允许玩家更快地购买第二个英雄或属性书。 |
| **商店优化** | 永久增加商店刷出**稀有装备**的概率。 | 星火精粹 | 提高高品质装备的获取效率。 |
| **图鉴解锁** | **解锁新的英雄类型、新的装备词条或新的基地升级功能**，并将其加入到下次游戏的随机池中。 | 星火精粹 | 增加游戏内容和构建（Build）的多样性。 |
| **精粹获取效率** | 永久提高每局游戏结束后获得的**星火精粹数量**。 | 星火精粹 | 加快整体元进度的推进速度。 |

## 🛡️ 英雄培养与属性系统

### 1. 英雄属性与成长

英雄不会自动升级，所有属性增长都来自购买和使用对应的**属性书**。

| 属性书 | 职业倾向 | 主要影响 (P) | 次要影响 (S) |
| :--- | :--- | :--- | :--- |
| **力量 (STR)** | 守护者/前排 | **生命值**、**物理防御** | 普攻伤害小幅提升。 |
| **敏捷 (AGI)** | 火力手/输出 | **攻击速度**、**移动速度** | 暴击率提升。 |
| **智力 (INT)** | 策术师/辅助 | **技能强度**、**能量恢复速度** | 元素抗性提升。 |
| **元素专精** | 所有英雄 | **元素穿透**，**增强对应元素技能伤害**。 | 提升特定元素抗性。 |

### 2. 英雄定位与技能

每个英雄拥有一个**元素专精**和**一个主动技能**（英雄**自动施放**，消耗能量）。

| 英雄定位 | 属性侧重 | 技能施放逻辑 (AI) | 核心技能设计 |
| :--- | :--- | :--- | :--- |
| **守护者** | 力量 | **低血量/怪物数量多时** 自动施放。 | **地裂壁垒/冰霜护甲**：在承受压力时保护自己或防线。 |
| **火力手** | 敏捷 | **技能冷却就绪时** 自动施放，目标选择**威胁度最高**的怪物。 | **过载射击/雷电弹丸**：持续稳定地输出高额伤害。 |
| **策术师** | 智力 | **怪物接近基地/友方英雄需要治疗时** 自动施放。 | **烈焰结界/冰冻陷阱**：在关键区域提供控制或范围伤害。 |

## ⚰️ 英雄阵亡与复活机制

### 1. 阵亡惩罚（战斗阶段）

* **状态：** 英雄进入**“阵亡”**状态，失去战斗能力。
* **战线空缺：** 英雄阵亡后，该防线将失去战力，增加怪物突破的风险。

### 2. 复活机制（波次间隙 / 暂停时）

| 复活选项 | 消耗与成本 | 复活状态 | 策略考量 |
| :--- | :--- | :--- | :--- |
| **付费即时复活** | 大量**金币**或**星火碎片**，且**费用递增**。 | $50\%$ 生命值复活。 | 快速恢复战力，但严重挤占经济。 |
| **缓慢等待复活** | **免费**，但需等待 **1-2 个波次**。 | 满生命值复活。 | 节省资源，但需面临长时间减员的压力。 |

## 🏃 战斗部署系统

* **波次间隙 (暂停状态)：** 玩家可以**拖动**英雄图标，将其部署到核心周围的指定**防御区域**。
* **战斗阶段 (实时全自动)：** 英雄在部署区域内**自动移动**、**自动攻击**、**自动施放技能**。
* **资源与掉落：** 怪物死亡时**实时掉落**金币和装备，但玩家需等待波次结束（暂停时）才能在商店中使用资源或拾取装备。

## 👹 怪物类型与战术应对

怪物引入了**“元素抗性”**和**“路径特性”**，迫使玩家混合部署不同元素的英雄。

| 怪物类型 | 特点 | 元素抗性 | 战术应对 |
| :--- | :--- | :--- | :--- |
| **影界低语者** | 基础小怪，数量庞大，血量低。 | **无** | 依赖范围伤害（火/雷）和高攻速的敏捷英雄清理。 |
| **腐蚀爬行者** | **速度极快**，目标直奔基地。 | **土/物理** | 必须依靠**冰系控制**或**位移技能**快速拦截。 |
| **以太巨兽** | **血量极高**的精英怪，移动缓慢，攻击力强。 | **火** | 依赖**雷系**或**物理穿透**进行单体爆发；需要守护者持续嘲讽。 |
| **混沌法师** | 远程单位，会周期性为其他怪物施加**元素护盾**。 | **智力系伤害** | 必须用**敏捷/力量**英雄快速突进或用单体爆发技能**打断**施法。 |
| **元素畸变体** | 由单一元素构成的怪物。 | **自身元素抗性极高**。 | 必须使用克制元素（如：火克冰，雷克土）或纯物理伤害。 |

## 🏰 基地升级系统：星火核心

基地的升级主要消耗金币（Gold）和稀有的**“星火碎片”**（Stardust Shards）。

### 1. 核心属性升级（全局被动增益）

| 升级项目 | 效果描述 | 资源消耗 | 策略价值 |
| :--- | :--- | :--- | :--- |
| **星火储备** | 永久增加基地核心的**生命值**上限。 | 金币 | 增加容错率。 |
| **能量汲取** | 永久增加所有英雄的**能量恢复速度**。 | 金币 / 星火碎片 | 英雄技能释放更频繁。 |
| **元素增幅** | 提高所有英雄**元素专精**带来的伤害加成。 | 金币 | 强化元素输出队伍。 |

### 2. 核心防御功能（主动技能/防御塔）

| 升级项 | 效果描述 | 解锁消耗 | 作用 |
| :--- | :--- | :--- | :--- |
| **净化冲击 (主动)** | 激活后，对全地图所有怪物造成一次低伤害，并短暂**击晕** $1$ 秒（每局限次数）。 | 金币 + 星火碎片 | 打断关键施法或在防线崩溃时救急。 |
| **治疗无人机** | 自动环绕基地，对范围内的**低血量英雄**周期性进行小额治疗。 | 金币 | 增强英雄的续航能力。 |

## ✨ 装备系统：词条与策略选择

英雄有**六个装备槽**。装备掉落具有随机的**主属性**和**随机词条**。

| 词条名称 | 效果描述 | 最佳适用英雄 | 策略价值 |
| :--- | :--- | :--- | :--- |
| **溅射** | 普攻有 $25\%$ 几率对目标周围小范围造成伤害。 | 敏捷（高攻速） | 增加清怪效率。 |
| **星火共鸣** | 释放技能时，额外回复 $5\%$ 能量。 | 智力（高施法频率） | 形成技能循环。 |
| **仇恨引导** | 每次受到伤害，增加英雄 $5\%$ 仇恨值，可叠加。 | 力量（守护者） | 确保坦克能稳稳拉住BOSS仇恨。 |
| **致残** | 暴击时，有 $50\%$ 几率使目标移动速度降低 $30\%$。 | 敏捷（高暴击率） | 辅助控场，减缓突破防线的快速怪物。 |
| **元素转换** | 普攻 $10\%$ 几率对目标施加**相反元素**的易伤效果。 | 所有英雄 | 克服怪物的元素抗性。 |
