## III. ⚔️ 统一战斗与信息界面 (RTS Bottom Panel Style)

核心设计：屏幕底部区域统一为信息面板。根据玩家点击的物体（基地、英雄、空地），以及当前的游戏状态（战斗中/暂停中），底部面板会切换功能。

### A. 实时战斗状态 (Combat State)

| 区域 | 位置/功能 | 内容与功能 |
| :--- | :--- | :--- |
| **顶部信息栏** | 屏幕顶部 | **基地核心状态**（中央上方）、**波次信息**（当前波次编号）、**资源显示**（金币/精粹）。 |
| **底部信息区 (主面板)** | 屏幕下方 $1/4$ 区域 | **功能随点击目标切换：** |
| **1. 点击空地** | 底部左侧：**已部署英雄**的小头像列表（显示血条、能量条）。 底部中央：**基地全局技能**（如：净化冲击）及其冷却时间。 底部右侧：**暂停/商店按钮**。 |
| **2. 点击英雄** | 底部左侧：被点击英雄的**详细属性**（STR/AGI/INT）和**装备槽位**（6个）。 底部中央：英雄的**基础信息**（名字、元素专精）和**技能图标**（显示冷却）。 底部右侧：**英雄状态**（如：阵亡倒计时）。 |
| **3. 点击基地** | 底部左侧：**基地属性/状态**（核心生命值百分比、防御无人机状态）。 底部中央：**基地升级/元进度**的**快捷入口**（点击后进入暂停状态）。 底部右侧：**暂停/商店按钮**。 |

### B. 💰 策略/商店界面 (Strategy Hub - 暂停状态)

当玩家在波次间隙自动暂停时，或手动点击 **[暂停/商店按钮]** 时，游戏进入此模式。

* **屏幕顶部和中部：** 显示地图和英雄部署区（英雄图标可拖拽）。
* **底部信息区：** 此时底部面板功能被**统一锁定**为策略和购买。

| 区域 | 内容与功能 | 重点设计 |
| :--- | :--- | :--- |
| **底部左侧** | **英雄面板/复活** | **可切换英雄**，查看详细属性，进行**装备穿戴/卸下**。阵亡英雄在此处进行**复活操作**。 |
| **底部中央** | **商店区** | **核心购买区域**，分为标签：**新英雄**、**属性书**、**随机装备**。 |
| **底部右侧** | **基地升级区** | **星火核心升级**（属性强化、无人机、共鸣炉）和**元进度**（永久升级）的详细面板。 |

---

这个设计确保了：

1.  **高效率：** 战斗中，信息总是在玩家视线可及的底部区域切换，无需频繁将视线移到屏幕角落。
2.  **RTS 经典体验：** 玩家可以通过点击单位来获取信息和状态，战斗界面的视觉中心保持在核心地图上。
